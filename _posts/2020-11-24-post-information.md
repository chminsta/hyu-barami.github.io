### 이미지 업로드 방법
```

이미지 이름은 원하는대로 해도 된다.
# Example
```

### HTML 작성 방법 
글 내에 html를 활용해 코드를 작성하고 싶다면, 이런식으로 쓸 수도 있다.

깃헙에서 이 글의 markdown 코드를 확인하여 어떻게 작성되었는지 확인해보자! 

<div>
    누르면 숫자가 증가하는 버튼
    <button id="counter">0</button>
    <script>
        cnt = 0;
        document.getElementById("counter").addEventListener("click",function(e) {
            cnt++;
            this.innerText = cnt;
        });
    </script>
</div>
